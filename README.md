# Relationships Notifier [![CodeFactor](https://www.codefactor.io/repository/github/Twizzer/relationships-notifier/badge)](https://www.codefactor.io/repository/github/Twizzer/relationships-notifier)

A powercord plugin that notifies you when someone removes you from their friends list, you are banned/kicked from a server or kicked from a group chat.

# Requirements

-  Powercord

# Installation

-  `git clone https://github.com/Twizzer/relationship-notifier.git`
